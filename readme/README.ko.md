# TianshangPeriodPal（天殇·月記）

## 개요

완전히 오프라인으로 작동하며, 로컬 암호화를 지원하는 오픈소스 월경 기록 및 관리 앱입니다. 제로 네트워크와 강력한 암호화를 통한 기술 아키텍처로 데이터가 항상 사용자 손 안에 머물도록 합니다.

## 기술 스택

- **언어/프레임워크**: Kotlin, Jetpack Compose (Material 3)
- **데이터베이스**: Room (로컬 SQLite, SQLCipher 암호화)
- **아키텍처**: MVVM + Repository
- **최소 SDK**: 28 (Android 9), 타겟 34

## 핵심 기능

- 월경 주기 기록 및 예측
- 배란기 및 가임기 예측
- 생리통 및 증상 분석
- 로컬 암호화 데이터 저장
- 생체 인식 앱 잠금
- 다국어 지원 (7개 언어)
- 데이터 백업 및 내보내기
- 휴지통 기능

## 개인정보 보호 기능

- 완전 오프라인 — 네트워크 요청 제로
- SQLCipher로 데이터베이스 암호화
- 지문/비밀번호 앱 잠금
- 모든 데이터가 기기에만 저장
