# TianshangPeriodPal（天殇·月記）

## 概要

完全オフラインで動作する、ローカル暗号化対応のオープンソース月経�記録・管理アプリです。ゼロネットワークと強力な暗号化による技術アーキテクチャにより、データは常にユーザーの手の中に留まります。

## 技術スタック

- **言語/フレームワーク**：Kotlin、Jetpack Compose（Material 3）
- **データベース**：Room（ローカル SQLite、SQLCipher 暗号化）
- **アーキテクチャ**：MVVM + Repository
- **最小 SDK**：28（Android 9）、ターゲット 34

## 主な機能

- 月経周期の記録と予測
- 排卵期と受孕期の予測
- 月経痛と症状の分析
- ローカル暗号化データストレージ
- バイオメトリックアプリロック
- 多言語対応（7言語）
- バックアップとエクスポート
- ゴミ箱機能

## プライバシー機能

- 完全オフライン — ネットワークリクエストゼロ
- SQLCipher によるデータベース暗号化
- 指紋/パスワードによるアプリロック
- データはデバイス上にのみ保存
