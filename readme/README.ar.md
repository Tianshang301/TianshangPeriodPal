# TianshangPeriodPal (تيانشانغ·يوه جي)

## نظرة عامة

تطبيق لتتبع وإدارة الدورة الشهرية، يعمل بالكامل دون اتصال بالإنترنت، مع تشفير محلي ومفتوح المصدر. من خلال بنية تقنية لا يمكن اختراقها (صفر اتصال بالشبكة، تشفير قوي)، تظل بياناتك في يدك دائمًا.

## التقنيات المستخدمة

- **اللغة/الإطار**: Kotlin، Jetpack Compose (Material 3)
- **قاعدة البيانات**: Room (SQLite محلي، تشفير SQLCipher)
- **البنية**: MVVM + Repository
- **الحد الأدنى لـ SDK**: 28 (Android 9)، الهدف 34

## الميزات الأساسية

- تسجيل وتوقع الدورة الشهرية
- توقع وقت التبويض والفترة الخصبة
- تحليل آلام الدورة الشهرية والأعراض
- تخزين البيانات المشفرة محليًا
- قفل التطبيق بالبصمة
- دعم متعدد اللغات (7 لغات)
- النسخ الاحتياطي وتصدير البيانات
- سلة المحذوفات

## ميزات الخصوصية

- يعمل بالكامل دون اتصال بالإنترنت — صفر طلبات شبكة
- تشفير قاعدة البيانات باستخدام SQLCipher
- قفل التطبيق بالبصمة / كلمة المرور
- جميع البيانات مخزنة على الجهاز فقط
